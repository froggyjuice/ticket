# 🎻 관현악단 자체 티켓팅 시스템

수동 예약 방식을 탈피하고, 우리 악단만의 디지털 티켓팅 및 스마트 팜플렛 시스템을 구축하는 프로젝트입니다.

---

## 개발 파이프라인

**Antigravity → GitHub → Vercel**

---

## 핵심 기능

- **간편 로그인** — 이름 + 전화번호 기반, 별도 회원가입 없이 예매 진행
- **좌석 배정** — 지정석 및 랜덤 배정, 단원 파트별 최적 추천 좌석 안내
- **디지털 티켓** — 저장·공유 가능한 이미지(jpg) 실시간 생성 (객원 초청용)
- **기간별 제어** — 기수별 비밀번호로 가족·친구·지인 예매 오픈 기간 차등 설정
- **스마트 팜플렛** — 실시간 수정 가능한 디지털 북, 유튜브·위키백과 링크 연동
- **메인 허브** — 공연 디데이, 포스터, 예매 바로가기, 연습 사진 갤러리
- **관객 방명록** — 모바일 청첩장 형태의 한줄평·기대평 작성 공간
- **수요 예측** — 종이 티켓·인쇄 팜플렛 사전 수요 체크로 제작 비용 절감

|------|------|
| 팀장 | 총괄 기획, 아키텍처 설계, API 명세, 배포(Vercel) 관리 |
